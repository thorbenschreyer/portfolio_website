# 🚀 Thorben Schreyer – Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and experience as a Frontend Developer.

## 🌐 Live Demo

🔗 https://thorben-schreyer.de

---

## 📋 About The Project

This portfolio website was built to present my work, technical skills, and professional background in a clean and interactive way.

Visitors can:

- Learn more about me
- Explore my technical skill set
- View completed projects
- Read references and testimonials
- Contact me directly through the contact form

The website is fully responsive and optimized for desktop and mobile devices.

---

## ✨ Features

- Responsive Design
- Multi-language support (German / English)
